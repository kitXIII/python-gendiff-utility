# A console utility that compares two configuration files and shows a difference.


## Requirements

* Python >= 3.11
* Poetry
* make

## Installation

### To install

1. clone repo
2. run
    - `$ make install`
    - `$ make build`
    - `$ make package-install`

### To uninstall

 run `$ make package-uninstall`
